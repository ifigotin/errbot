errbot.py2conv module¶

errbot.py2conv.convert_to_python2()[source]¶

Convert errbot source code (which is written for Python 3) to Python 2-compatible code (in-place) using lib3to2.

errbot.py2conv.walk_lib3to2_input_sources()[source]¶

Recursively walk through lib3to2 inputs, returning all the files detected while respecting the exclude list.
